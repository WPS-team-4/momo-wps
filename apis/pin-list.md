## Pin 생성

### method

POST

### body
	
#### A. 지도에서 장소를 선택해서 핀을 만드는 경우
- pin  
	- pin_name
	- map
	- pin_color
- place  
	- lat
	- lng


```
{
	"pin": {
		"map": "5",
		"pin_name": "CafeB",
		"pin_color: "0,0,0"
	},
	"place": {
		"lng": "126.9760823",
		"lat": "37.5667701"
	}
}
```

#### B. 장소 검색결과에서 장소를 선택하여 핀을 만드는 경우
- pin  
	- pin_name
	- map
	- pin_color
- place  
	- place_id
	- name
	- address
	- lat
	- lng


```
{
	"pin": {
		"map": "5",
		"pin_name": "CafeB",
		"pin_color: "0,0,0"
	},
	"place": {
		"place_id": "ChIJmc7grvKifDURHm9YxvGM9Rg",
		"name": "카페비",
		"address": "3-7 Jeong-dong, Jung-gu, Seoul, South Korea",
		"lng": "126.9760823",
		"lat": "37.5667701"
	}
}
```
